Usando etiquetas Flex Box, Listas y enlaces en HTML

Proposito del sitio

El proposito de este sitio tiene como fin explicar los conceptos basicos de flexbox, listas y enlaces y asi comptender y posteriormente usarlos en HTML, asi como tambien poder realizar un correcto diseño de una pagina web aplicando las difrentes etiquetas y estilos.

Introduccion

En este trabajose pretende realizar un correcto diseño de pagina web alicando los defrentes estilos y etiquetas de html, asi como tambien se pretende dar unas definiciones de flexbox, listas y en laces en html.
