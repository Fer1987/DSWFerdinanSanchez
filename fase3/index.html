Usando etiquetas Flex Box, Listas y enlaces en HTML

Proposito del sitio

Este trabajo tiene como propósito explicar los diferentes conceptos de flexbox, listas y enlaces en html y así poder poner en practica todo lo comprendido para poder realizar un correcto diseño de nuestro sitio aplicando las diferentes etiquetas y estilos en html.

Introduccion

Por medio de este trabajo se pretende que los lectores entiendan de forma clara los diferentes conceptos de etiquetas Flex Box, Listas y enlaces en HTML, para que tengan una guía de como podemos realizar un diseño de una pagina web.
